PST Viewer

Folder View

    Welcome to the Vacasa PST viewer!

    This app will let you quickly view the contents of a PST or OST email archive.

    No file data or metadata is sent to our servers - all of the processing is done by your machine in the web browser.

    For questions or comments, please email itsupport@vacasa.com.

    To get started, drag a PST or OST file to this browser window, or click below to select one.

    Recent Changes:

    - Added ability to open OST files (including Outlook 2016 OST format)

    - Updated entire interface to include 2019 java updates
